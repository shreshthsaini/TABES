Trajectory-Aware Backward-on-Entropy Steering for Masked Diffusion Models

TABES is a training-free guidance method that fixes trajectory lock-in by pairing adaptive forward entropy steering with backward uncertainty correction.

ArXiv Code (coming soon) Read the Method Blog

Why TABES?

Problem: trajectory lock-in

Standard entropy minimization can commit early to wrong tokens. As denoising proceeds, alternatives collapse and decoding quality drops.

Core idea: steer both directions

TABES combines confidence growth in the forward chain with uncertainty-aware correction from a reversed trajectory estimate.

Practical: no retraining

The method is drop-in for masked diffusion LMs and targets better outputs with controlled computational overhead.
